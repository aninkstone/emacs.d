# Evil Rails

rails.vim for Emacs. Based on:

- [evil-mode](http://gitorious.org/evil)
- [Rinari](http://rinari.rubyforge.org/)

# Supported commands

    :Rcontroller
    :Rmodel
    :Rspec
    :Rhelper
    :Rmailer
    :Rmigration
    :Rstylesheet
    :Rjavascript
    :Rfile
    :Rsass
    :Rfeature
    :Rcell
